# Trademarks

This repository contains official Obsura brand identifiers.

## Current Status

Obsura does not yet have a finalized standalone trademark policy.

Until one is published, treat the `Obsura` name, logos, marks, and official
brand identifiers as protected project identity rather than generic design
assets.

## Practical Rule

Reasonable factual reference is fine. Confusing brand reuse is not.

Acceptable examples:

- referring to Obsura by name in documentation or compatibility notes
- linking to official Obsura repositories
- using unmodified official assets inside official Obsura repositories
- showing official branding when accurately pointing to the project itself

Avoid:

- presenting a fork, service, package, or derivative work as an official Obsura
  release when it is not
- modifying an official mark and continuing to present it as the official mark
- using the `Obsura` name or visual identity in a way that implies endorsement
  without permission

## Future Refinement

A more detailed policy may be added later. When that happens, this file should
be updated alongside `obsura-standards`.
