# Messaging

This document provides the official description set for Obsura and the message
themes that should repeat across repositories and public surfaces.

See also: [Verbal Identity](verbal-identity.md), [Taglines](taglines.md),
[Website And Product Copy](website-and-product-copy.md)

## One-Line Description

Obsura is an open-source, self-hosted redaction platform for safely sharing
screenshots, text, logs, and technical files.

## Short Description

Obsura helps users detect, review, style, and securely remove sensitive
information before they share screenshots, logs, technical text, and other
operational content.

## Medium Description

Obsura is an open-source, self-hosted redaction workbench built for real
sharing workflows. It helps users detect sensitive information, review proposed
changes, apply consistent redaction or replacement styles, and export content
that is safe to share.

## Long Description

Obsura is an open-source, self-hosted redaction platform focused on screenshots,
images, pasted text, logs, and structured technical content. It is designed for
people who need to hide usernames, hostnames, IP addresses, domains, secrets,
or internal identifiers quickly without losing confidence in the result. Obsura
emphasizes review-first workflows, practical daily-use speed, visual redaction
styling, and safe export rather than vague automation claims.

## Positioning Statement

Obsura helps people detect, review, style, and securely remove sensitive
information before they share operational or technical content.

## Brand Promise

Obsura aims to make safe sharing feel controlled, reviewable, and fast.

## Message Pillars

### Review Before Release

Obsura favors review-first workflows over opaque automation-first claims.

### Built For Real Technical Content

Obsura is shaped by screenshots, logs, pasted terminal output, configs, and
other daily operational artifacts.

### Self-Hosted And Open

Obsura should feel compatible with teams and individuals who care about control,
inspectability, and local ownership.

### Precision, Not Theater

Obsura focuses on exact redaction workflows, clear outcomes, and serious
software quality rather than security cliches.

## Claims To Avoid

Avoid unsupported claims such as:

- best-in-class
- enterprise-ready for everyone on day one
- fully autonomous redaction
- supports every file type
- military-grade privacy
