# Sample UI UX References

This directory contains third-party or external UI reference images used to
extract the current Obsura visual system.

These files are:

- reference-only
- not official Obsura product screenshots
- not official Obsura marketing compositions
- not reusable as brand exports or runtime assets

Use them to study:

- layout patterns
- component styling
- surface hierarchy
- spacing and radius language
- color balance and accent restraint

The adopted outcome of this reference set is documented in:

- [`../../docs/brand-system-freeze.md`](../../docs/brand-system-freeze.md)
- [`../../docs/visual-direction.md`](../../docs/visual-direction.md)
- [`../../docs/design-foundations.md`](../../docs/design-foundations.md)
